Part 4: Modeling

# The variables: Audience score, critics_score and imdb_rating are included in the full model. 

# Other variable were excluded because they were meaningless and for information use only. Audience score and critics_score and imdb_rating were selected because the research question was to address if there association between audience score and critic score.

In order to model the relationship, a linear model was fit to determine the relationship with the following results:

 

lmfit <- lm(movies$imdb_rating ~ movies$audience_score + movies$critics_score)

Part 5: Prediction

# In this part, the previous model is used to make prediction above a particular movie.

# A movie: Wonder Woman is selected , with an average imdb rating of 7.5. To do this first, we need to create a new data frame for this movie.

newmovies <- data.frame(title ="Wonder Woman",  = 7.5)

predict(lmfitlast, newmovies)

  1 

7.07936 

 

# This model predicts that the imdb rating for Wonder Woman is 7.5 and it is a good prediction. A prediction interval with confidence of 95% level is constructed to show the uncertainty based on this prediction.

 

predict(lmfitlast, newmovies, interval = "prediction", level = 0.95)

 

      fit      lwr      upr

1 7.07936 6.008107 8.150614

 

# The confience interval suggests that there is a 95% confidence that the movie Wonder Woman having an average of 7.5 rating will receive an imdb rating between 6.00 and 8.15.

 

 

 

Part 6: Conclusion

# This project is conducted using a statistical dataset of movies to analyze the different variables and determine any significant relationship among the selected variables. So, the variables imdb rating (Y) and audience score (X) being the main explanatory variable is selected for the analysis. The plots and regression analysis suggest that the audience score is a significant predictor of imdb rating at 5% significance and 95% confidence level. The model is also used to predict the imdb rating for another movie. It implies that movie producers and directors should consider audience rating when making a new film as it can be a success factor for their movies. This research did not considered the role of days and months in which movies may have high rating. Like when people have more free time in summer vacations. So, future research can look into these factors for better analysis and prediction.
